#  First App - Name List Manager

A simple web application that allows users to add names, display them in a list, and delete them.  
Built using **HTML, CSS, and JavaScript** with localStorage support.

---

##  Features

- Add names dynamically
- Display names in a list
- Delete names individually
- Data saved in browser (localStorage)
- Simple and clean UI

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---


##  How it works

- User types a name
- Clicks "Add"
- Name is saved in localStorage
- List is updated automatically
